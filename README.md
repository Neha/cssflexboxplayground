# flexbox playground
This is a demo to showcase how the CSS Flexbox works.

## Why?
Many times we get confuse to understand how a particular CSS property works.
This playground is an attempt to help to visualize the same.

## Tech Stack?
- HTML
- CSS
- JavaScript
- For build : ParcelJS
